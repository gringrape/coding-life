# 쇼핑몰

## 기능

### 주문

- [ ] 상품 주문

### 상품

- [x] 상품 목록
- [x] 상품 상세

### 회원

- [ ] 회원 가입
- [ ] 로그인

## 관리자 서버

- 상품 등록
- 상품 수정
- 상품 삭제

## 회원가입

### 회원 정보 입력

- 이름, 이메일, 주소, 비밀번호, 전화번호

### 도메인

- 이름, 이메일, 주소, 비밀번호(해시), 전화번호

### 테이블

- 이름, 이메일, 도시, 상세주소, 비밀번호(해시), 전화번호
